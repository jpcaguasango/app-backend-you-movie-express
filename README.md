## Environment

Create the .env file

# The movie db

- THE_MOVIE_BEARER_TOKEN=
- THE_MOVIE_BASE_URL_API=

# Server config

- PORT=5000
- PATH_URL=http://localhost:5000
- SERVER_UP=the server is running on the port

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run dev

# watch mode
$ yarn run start
```

## Test

```bash
# unit tests
$ yarn run test
```
